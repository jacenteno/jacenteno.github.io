iOsSig - Cliente Móvil para ERP

Aplicación nativa para iOS construida enteramente con SwiftUI, que funciona como un cliente móvil para el sistema ERP empresarial "SIG". La app empodera a los empleados de las empresas suscritas, permitiéndoles gestionar operaciones clave como ventas, inventario y clientes directamente desde su iPhone o iPad.

Funcionalidades Clave:

Tecnologías Utilizadas: SwiftUI, Combine, Arquitectura MVVM.

Proyectos en Django

Proyecto: DigiPromo - Plataforma de Gestión de Promociones

DigiPromo es una solución web integral desarrollada en Django para la creación, gestión y aplicación de promociones y ofertas. La plataforma está diseñada para optimizar campañas de precios tanto en puntos de venta (POS) físicos como en plataformas de e-commerce, a través de un panel de administración intuitivo y una potente API RESTful.

Arquitectura y Componentes Clave

  • Panel de Administración: Centro de control para la gestión de promociones, construido con Django Admin y personalizado con el moderno tema Unfold para una experiencia de usuario superior.
  • API RESTful: Núcleo del sistema para integraciones. Desarrollada con Django REST Framework (DRF), proporciona endpoints seguros y documentados con Swagger (drf-yasg) para la consulta y aplicación de promociones en tiempo real.
  • Frontend para Clientes: Interfaz web para la visualización de ofertas, construida con plantillas de Django y estilizada con Tailwind CSS para un diseño moderno y responsivo.

Características Técnicas Destacadas

  • Flujo de API Optimizado para POS: Diseño de un flujo de comunicación eficiente que minimiza las llamadas de red, permitiendo a los sistemas POS cachear las promociones del día para una validación local y ultrarrápida durante el escaneo de productos.
  • Seguridad Robusta: Autenticación de API basada en tokens con Simple JWT. Implementación de medidas de seguridad como Throttling (limitación de tasa) para proteger los endpoints contra abusos.
  • Tipos de Promoción Flexibles: El sistema soporta lógicas de negocio complejas, como ofertas "Lleve X, Pague Y" y promociones de "Precio Fijo", con una arquitectura de modelos extensible.

Stack Tecnológico

Python Django Django REST Framework PostgreSQL Tailwind CSS JWT Swagger (drf-yasg)

Plataforma de Gestión Comercial (SIG)

Sistema integral que centraliza la gestión comercial y la relación con clientes. La plataforma incluye módulos clave como un motor de Promociones , un sistema de Lealtad por puntos, y un procesador de datos asíncrono (con Celery) para integrar ventas de sistemas externos.

Toda la lógica de negocio se expone a través de una API RESTful robusta, que da servicio a múltiples clientes: una app móvil (React Native/Kotlin), un kiosko de autoservicio y herramientas de análisis de datos.

Tecnologías Destacadas:

  • Backend: Python, Django, Celery, Redis
  • API: Django REST Framework (DRF)
  • Frontend & Móvil: React Native, Kotlin (Android), JavaScript
  • Base de Datos: PostgreSQL (con vistas y queries complejas)
